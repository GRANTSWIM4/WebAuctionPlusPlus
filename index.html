This is a redesigned interface for WebAuction+. WebAuction+ is based off of WebAuction.

Install

  1. First, download the latest web files and the latest JAR file.
  2. Next, create a MySQL database and user. Do not forget what you put in!
  3. Drop the JAR file from the second file you downloaded into your plugins folder and reload the server. You can now delete that file.
  4. Next, add the MySQL username and password to the configuration file the plugin generated.
  5. Now, open up the first file you downloaded. Edit the config.php with your MySQL details.
  6. You should now upload the files to your web server. Reload your Minecraft server again.
  7. Yay! You should now do /wa password [password] to set your password, and then log in online!
  8. You should now read WebAuction+'s documentation to set it up!
